Objects and classes of the fantomEngine


The following classes/objects ship with fantomEngine and will be created directly:
  • ftAStar   A* path finding
  • ftBox2D   Box2D integration
  • ftEngine   The engine class
  • ftFont   Bitmap font, based on the EZGui file format
  • Misc functions
  • ftGui   GUI class
  • ftHighScoreList   Highscore list
  • ftImageMng   fantomEngine's own image manager class
  • ftKeyMapping   Key mapping support class
  • ftLayer   Layer group objects together to handle them in one batch
  • ftLocalization   Multilanguage support class
  • Misc functions   Misc helper functions
  • ftObjAnimMng   Object animation manager class which handles the various animation of an object
  • ftObject   Object class which can be images, primitives, text, collision zones
  • ftRGBA   Class that provides color objects
  • ftScene   Class that adds scenes to fE
  • ftSound   Sound class
  • ftSpriteAtlas   Class to manage sprite sheets more easily
  • ftMapObj   Class that stores Tiled map objects
  • ftTimer   Timer class which handles ftEngine and ftObject timer
  • ftTrans   Transition class
  • ftVec2D   Class that provides 2D vector objects
  • ftPath and ftWaypoint   Path and waypoint handling

The next classes/objects are used by the fantomEngine internally or have more of a support character:
  • ftHighScoreEntry   Highscore list entry
  • ftSwipe   Swipe class
  • ftTransEntry   Transition entry class
  • tPoints   Helper class to store a 2D vector definition
  • tLine2D   Helper class to store a 2D line definition
  • ftContactListener   Helper class for Box2D collision callbacks
  • ftMapTile   Helper class for ftObjects that represent a tile of a tile map
  • ftTileMap   Helper class for ftObjects that represent a tile map
  • ftAnimFrame   Helper class for ftAnim that represent a frame of an animation
  • ftMarker   Helper class for ftPath that represent a running object marker on a path
