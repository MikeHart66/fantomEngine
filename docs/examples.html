Example scripts


The fantomEngine module ships with some example scripts that show you the usage of a certain feature. They are located inside the examples folder located in your fantomEngine module installation.

The following topics can help you create your app with fantomEngine

Basics

Base script
To see a skeleton script for the fantomEngine, have a look at the BaseScript.monkey script.

Input

Keymapping
Sometimes you need to support different input schemes for your games. The KeyMapping.monkey shows you how you can handle this.

Swipe/Gesture detection
To find out how to use the swipe gesture detection, check out the SwipeDetection.monkey script.

Layer managment

Scrolling layers
If scrolling layers is still a black book for you, then maybe this script from fellow fantomEngine user Douglas Williams will help you... ScrollingLayers.monkey

Localization

Multiple languages
If you need a simple way to display text in different languages then have a look at this script: Localization.monkey.

Misc

Jump 'n' example
There are several ways to create a jump 'n' run game in fantomEngine. This script by Douglas Williams shows you one way: RunJump.monkey.

Multi class projects
If you wonder how to split your project into several classes, have a look here: MultiClass.monkey.

Update objects
To see how you can update your objects, have a look here: UpdateObjects.monkey.

Objects

Appearence

Set the blend mode
Sometimes you need to see the blendmode of an object. For this you can study the BlendMode.monkey script.

Let an object flash
If you need to draw attention to an object, let it flash like in this script (extended by Douglas Williams)... FlashObject2.monkey

Behaviour

Object movement
If you want to find out how to move objects and handle them in the OnObjectUpdate event, have a look at the ObjectMovement.monkey script.

Collision detection

Collision checks between objects
If you need to check for collisions between your objects, look at this example: Collision.monkey.

Object creation

Animate an object
Besides using an image strip like you do in Mojo, you can build animations in fantomEngine by adding several images to an object. Have a look at this example: AnimObject.monkey.

Copy an object
When you need to copy an object, have a look at the CopyObject.monkey script.

Drawing Primitives
fantomEngine supports different drawing primitves and the CreatePrimitives.monkey script by Douglas Williams tell you how.

Extend an object
When you need to extend an object, have a look at the CustomObject.monkey script.

Load images
When you need to load images, it is good to have a look at this example: LoadImage.monkey.

Object pooling
If you need object pooling, look here: PoolingObjects.monkey.

Remove Objects
If you need to remove objects, have a look at the RemoveObject.monkey script.

Data Storage

Storing user defined data
Sometime you need to store extra information inside an ftObject. For this, have a look at the DataStorage.monkey script.

Object hierachy

Parent and childs
If you need a parent child relationship of your objects, have a look at the ParentChild.monkey script.

Sort Objects
If you need to sort objects in layers, have a look at the SortObjects.monkey script.

Pathfinding

A* path finding
If you need to utilize A* path finding in your game, then the Pathfinding.monkey script is the place to look at.

Physics

Box2D physics
There are several examples on how to integrate Box2D physics with fantomEngine. Here is a short list...
  • Box2DIntegration.monkey
  • JuggleSoccerBall.monkey
  • MouseJoint.monkey
  • Raycasting.monkey
  • RevoluteJoint.monkey


Scenes

Scene management
Sometimes a game is divided into scenes/rooms. To have a go at this feature, look at this example: SceneCreation.monkey.

Spritesheets

Using Spritesheets for animated objects
If your animated sprites are bundled inside sprite sheets, please look at the SheetAnim.monkey script.

Texture atlas / Sprite sheets
Sprite sheets are a great way to preserve video memory and also speed up the render process. To use texture atlas created by the tool TexturePacker, check out the TexturePacker.monkey script.

Text

FontMachine
If you want to use FontMachine fonts, you don't need to do anything special: FontMachine.monkey.

Multiline text
To display multiline text, have a look at the MultilineText.monkey script.

Tilemaps

Endless maps


For something like an endless runner game, have a look at this script to see how you can approach this: EndlessMap.monkey.

Dynamic created maps


If you need to build a map dynamically, have a look at this script: DynamicMap.monkey.

Platformer game


To see how you could do collision checks against a Tiled platformer level, have a look at this script: Platformer.monkey.

Maps created by Tiled

Orthographic maps
For big tile maps, created by the tool Tiled, have a look at the Tiled.monkey script. It even handles map objects now.

Displaying isometric TILED maps
To display and use an isometric map which was build in Tiled, study this example script: TiledIsometric.monkey.

Using multiple tile sheets in on map
If your map uses multiple tile sheets, look at this example: MultiTileSets.monkey.

Timer

Using Timer
Timer make your game tick. Look at this script: Timers.monkey.

Transitions

Transitions
If you want to use transitions with tweeing, then have a look at the ObjectTransition.monkey script.

Waypoints

Paths and waypoints
To use paths/waypoints and how to move an object along a path, have a look at the Waypoints.monkey script.
